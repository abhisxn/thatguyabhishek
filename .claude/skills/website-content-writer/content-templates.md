# Content Templates & Destination Specs

Reference for outline structures and destination-specific formatting rules.

---

## Outline Templates (Step 2)

### LinkedIn Post
```
HOOK:  [opening line — punchline or provocation]
BEAT 1: [setup or human moment]
BEAT 2: [the reframe or pattern]
BEAT 3: [specific example or proof point]
CLOSE: [real question or landing thought — not rhetorical bait]
```

### LinkedIn Thread
```
1/ HOOK: [standalone opening beat — must work without context]
2/ [beat — setup or context]
3/ [beat — the argument builds]
4/ [beat — the turn or evidence]
5/ [beat — the implication]
6/ [beat — the key insight]
7/ CLOSE: [landing — not a CTA]
```

### LinkedIn Carousel
```
POST COPY: [self-contained argument — full point, no slides needed]
SLIDE 1:   [hook / title slide]
SLIDE 2:   [beat — one idea]
SLIDE 3:   [beat — one idea]
SLIDE 4:   [beat — one idea]
SLIDE 5:   [beat — one idea]
SLIDE 6:   [closing slide — the question]
Note: Post copy carries the full argument. Slides amplify, not summarise.
```

### LinkedIn Poll
```
QUESTION:  [the premise — specific, not vague]
OPTION A:  [clear position]
OPTION B:  [clear counter-position]
OPTION C:  [nuanced middle if needed]
OPTION D:  [wildcard or "none of the above" if needed]
CONTEXT:   [2–3 lines below the poll — why this matters]
```

### Website: Article
```
TITLE:    [direct, specific — not clickbait]
LEDE:     [the point in 2–3 sentences — full argument compressed]
SECTION 1: [name] — [one line: what this argues]
SECTION 2: [name] — [one line: what this argues]
SECTION 3: [name] — [one line: what this argues]
COUNTER:  [the honest counter-consideration]
CLOSE:    [one thing the reader should think or do differently]
SEO PHRASE: [natural target phrase this would rank for]
```

### Website: Thinking Note
```
TITLE:    [direct, opinion-first]
OPENER:   [the point, immediately — 1–2 sentences]
BODY:     [one tight argument — 3–4 beats, no sections]
CLOSE:    [provocation or open question]
```

### Website: AI Musings (Experiment write-up)
```
OPENER:   [the observation — not "I built a thing"]
WHAT:     [what it is and what it does — one sentence]
WHY:      [the design thought or question behind it — one sentence]
HOW:      [brief — what was interesting to build, not a tutorial]
CLOSE:    [a real question or provocation — not "link in bio"]
```

## Destination Specs (Step 3)

### LinkedIn Post
- 150–300 words
- Single scrollable read — no headers, no bullets unless the format demands it
- Line breaks are intentional — short paragraphs, breath between thoughts
- Max 3 hashtags, only if genuinely relevant
- No "I'm excited to share", no "Thoughts?", no "Let me know in the comments"
- End with: `[Suggest: native image / no image / video / carousel slide]`

### LinkedIn Thread
- 5–7 beats
- Beat 1 = standalone. Someone should be able to repost without context.
- Each beat under 280 characters ideally, max 400
- Numbered: 1/ 2/ 3/ etc.
- No CTA at the end. The landing is the close.

### LinkedIn Carousel
- Post copy first — self-contained, full argument, reads without slides
- Slide outline: one idea per slide, one job per slide
- Closing slide always carries the question
- Post copy: 150–250 words
- Slides: headline + 1–2 supporting lines each
- **Self-contained carousel rule:** Key insights must appear in post copy, never only in slides.

### LinkedIn Poll
- Question is specific, not vague
- Options are genuinely different positions — not "agree / disagree / both"
- Context post (2–3 lines) explains why the question matters — not engagement bait

### Website: Article
- 800–1500 words
- Lede = 2–3 sentences, full point upfront
- Max 3 subheadings — not a listicle
- One natural SEO phrase woven in — not stuffed
- Ends with one concrete thing the reader should think or do differently

### Website: Thinking Note
- 200–400 words
- Opinion-first — the point in the first sentence
- No subheadings — single flowing argument
- Ends with a question or provocation

### Website: AI Musings
- 300–600 words
- Opens with the observation, never "I built a thing"
- Names what it is + does in one sentence; the design thought behind it in one sentence
- Includes: link to the experiment
- Ends with question or provocation
